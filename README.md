The Book Shelf Management System is a lightweight web application developed using MongoDB, Express.js, and Node.js with EJS as the frontend. The system allows users to add, view, edit, and delete books efficiently. 
It provides an easy interface for managing book details such as title and author, while MongoDB ensures secure and reliable data storage.
